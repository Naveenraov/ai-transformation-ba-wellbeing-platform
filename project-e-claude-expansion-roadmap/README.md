# Project E — Claude Expansion Roadmap
### From One HR Chatbot to Organisation-Wide Claude Adoption
### Spectrum.Life | AI Transformation Partner Portfolio | Naveen Rao V | April 2026

---

## What This Project Is

A structured, sequenced, 12-month roadmap for expanding Claude from Spectrum.Life's single live deployment — the MS Teams HR FAQ chatbot — to full organisation-wide adoption across all 8 operational departments.

This project directly addresses Stephen Costello's vision: to move from one use case to Claude embedded in every team, with 40-60% of all operational tasks automated or AI-assisted.

---

## The Starting Point

Spectrum.Life has one live Claude deployment today. Stephen built it, Stephen runs it, Stephen is the organisation's Claude SME. That is not a sustainable position for a CEO of a company targeting €100M in revenue by 2028.

This roadmap transfers that capability from one person to the entire organisation — in a way that is governed, clinically safe, and commercially measurable.

| **5 — Agentforce** | Months 9–12 | 3 Agentforce agents connected to Salesforce DHOS | 40%+ tasks automated · 400+ hrs/week recovered |

---

## The Three-Layer Framework

```
LAYER 3 — INTELLIGENCE
Agentforce + Claude API — autonomous multi-step agents connected to Salesforce DHOS

LAYER 2 — ADOPTION
Claude Projects (dept memory) · Claude Skills (governed templates) · Claude Artifacts (document generation)

LAYER 1 — FOUNDATION  ← Must come first
Governance framework · Prompt library (CRAFT) · AI enablement training
```

Layer 1 must come first. Without governance, Layer 2 creates inconsistent outputs and ungoverned data handling in a clinical environment. Foundation before adoption. Adoption before intelligence.

## Governance Principles

Every phase of this roadmap is governed by:

- **Clinical boundary** — Claude assists and drafts. It never makes clinical decisions. Human-in-the-loop at every clinical step.
- **GDPR Article 9** — Zero clinical special category data in Claude without DPA and consent mapping confirmed in advance.
- **EU AI Act alignment** — Internal tools assessed against the same high-risk criteria applied to Cara, Spectrum.Life's CE-certified AI companion.
- **Audit trail** — Every Claude-generated output reviewed by a human before action. All approvals logged.
